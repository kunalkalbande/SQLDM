How to use Help

This Help system can be accessed either through the Help menu or by pressing F1 on your keyboard while on the window you would like more information.

To search for topics that explain the specific information you need, select the Search All Help tab, type in the keyword in the space provided, and click Search for a list of all the topics that contain the keyword you entered.

To print the SQLdm Help topics:

  1. Select the top-level node in the table of contents and click the Print button.
  2. Check the Print the selected heading and all subtopics box to open your print dialog.
