8.6 Release Notes

SQL diagnostic manager (SQLdm) provides an unprecedented level of diagnostic information on the health, performance, and status of SQL Server instances across your environment. You can view, diagnose, and report on critical performance statistics from a central point of control.

Using a unique agent-less architecture, SQLdm gathers diagnostic information in real time, keeping database administrators and managers informed by providing customized alerts. Easy to install and use, SQLdm is an indispensable SQL Server management tool that benefits both experienced and junior database administrators.
