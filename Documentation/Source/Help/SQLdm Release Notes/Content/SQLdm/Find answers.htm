View help topics

SQLdm Help supports the following Web browser minimum requirements:

  • Internet Explorer 8
  • Mozilla Firefox 4
  • Google Chrome 6

Print help topics

You can print a help topic by selecting the topic in the table of contents, and then clicking the Print this topic icon on the right side of the Help window.

Use the PDF if you want to print more than a single topic. To print, click the PDF tab, and then click the Print file button. In the Print dialog box, select the pages you want to print before clicking Print.
