Get file group information

The File Groups tab of the Files view displays the selected database. Click a group on the File Groups tab to populate the File Group Usage chart. Select the drop-down list above the chart to choose a chart type:

  • Current Megabytes
  • Current Percent
  • Potential Megabytes
  • Potential Percent

TIP

Select multiple databases and multiple file groups to compare how they are used. This can help you quickly spot issues with your file usage.

Access the File Groups tab

You can open the File Groups tab of the SQLdm Databases Files view by selecting the appropriate SQL Server instance, clicking Databases > Files, and then clicking the File Groups tab.
