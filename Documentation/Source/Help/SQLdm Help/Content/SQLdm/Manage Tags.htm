Manage tags

The Manage tags window allows you to Add, Edit and Remove Tags. For more information on tags and their functionality, see Working with tags.

TIP

This window is available only to SQLdm Administrators.

The Manage Tags window shows a list of all existing tags. From here, you can:

  • Modify the list of servers, custom counters, and permissions associated with a tag
  • Create a new tag
  • Edit an existing tag
  • Remove a tag (Removing a tag does not delete items associated with that tag.)

Access the Manage Tags window

To access the Manage Tags window:

  1. Select File.
  2. Select Manage Tags.
