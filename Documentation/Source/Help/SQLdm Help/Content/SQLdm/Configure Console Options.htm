Configure console options

You can specify a number of console options in the Console Options window that affect overall product behavior. To access the Console Options window, select Tools > Console Options from the SQLdm Console toolbar.

The following tabs are available on the Console Options window:

General

The General tab allows you to set how often the console refreshes data, hide the Console when minimized, select how SQLdm manages chart data, and choose what colors you want SQLdm to use.

Alerts

The Alerts tab allows you to limit the number of historical alerts displayed in the Alerts view, which can improve the performance of the view when there is a large number of alerts.

Notifications

The Notifications tab allows you to configure how SQLdm notifies you when alert data crosses a threshold.
