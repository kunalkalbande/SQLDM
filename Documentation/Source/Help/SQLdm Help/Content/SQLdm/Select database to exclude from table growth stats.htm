Select a database to exclude from table statistics collection

SQLdm allows you to exclude non-essential databases from table growth statistics collection. Excluding a database from collection avoid clutter and unnecessary growth in your Repository. To exclude a database, select the appropriate database from the available list, and then click OK. You can exclude more than one database at a time.

Access this list of databases for exclusion

If you want to exclude one or more of your databases from table statistics collection, access the Server Properties dialog for that monitored SQL Server instance by right-clicking the instance and selecting Properties. Click the Table Statistics tab, and then click Select Databases to Exclude.
