Manage your SQL Server instances

The Manage Servers window (File > Manage Servers) allows you to see all the SQL Server instances monitored by SQLdm. You can:

  • Add new SQL Server instances to SQLdm
  • Edit the SQLdm configuration for a particular SQL Server instance
  • Remove a SQL Server instance from SQLdm
  • Access the VM Configuration window to associate your monitored virtual SQL Server instances to a host machine
  • Apply an existing alert template to a server or group of servers
  • Test the connection status of your SQL Server instances
