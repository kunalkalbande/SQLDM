Monitor SQL Server configuration settings

The Configuration view allows you to view the SQL Server configuration settings for the selected SQL Server instance. When diagnosing problems with SQL Server performance it is helpful to see how it is configured. In addition, SQLdm offers the ability to edit some configuration settings, depending on the option selected.

Access the Configuration view

You can open the Configuration view of the SQLdm Overview tab by selecting the appropriate SQL Server instance, and then clicking Overview > Configuration.

To edit a configuration setting:

  1. Select an option from the list.
  2. Click the Edit Value hyperlink in the Details pane.
  3. Enter the new value.
  4. Click OK.
