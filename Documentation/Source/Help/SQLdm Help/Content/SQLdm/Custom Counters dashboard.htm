Custom Counters panel

The Custom Counters panel displays the statistics for a selected custom counter on a monitored SQL Server instance. At least one custom counter must exist on the selected instance for this panel to display any data. Custom counters allow you to include more performance metrics than are provided by SQLdm. For additional information about custom counters, see how to use custom counters to track metrics.

Any alerts affecting the Custom Counters panel depend on the selected custom counter and alert threshold settings.
