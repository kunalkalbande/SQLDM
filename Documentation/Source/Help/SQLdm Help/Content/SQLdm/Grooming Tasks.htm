Groom your tasks

Grooming allows you to limit the amount of tasks displayed on your SQLdm Console. You can set the To Do List retention period to a length you find convenient. Note that beginning with SQLdm 7.0, the To Do feature is available only to users who had To Do activated in an earlier 6.x version. This version of SQLdm does not support this feature for new users.

TIP

If you use the list as a knowledge base, you might set the retention period to an extended length of time. If you use the To Do List as a daily reminder, you might set the retention period to two days.

To set the To Do List retention period:

  1. Select Tools > Grooming Options.
  2. Enter the retention period.
  3. Click OK.
