Wait type category - Lock

The Lock wait type category includes all the waits that are caused when a task is waiting to acquire a lock, including the following:

  • LCK_M_BU
  • LCK_M_IS
  • LCK_M_IU
  • LCK_M_IX
  • LCK_M_RIn_NL
  • LCK_M_RIn_S
  • LCK_M_RIn_U
  • LCK_M_RIn_X
  • LCK_M_RS_S
  • LCK_M_RS_U
  • LCK_M_RX_S
  • LCK_M_RX_U
  • LCK_M_RX_X
  • LCK_M_S
  • LCK_M_SCH_M
  • LCK_M_SCH_S
  • LCK_M_SIU
  • LCK_M_SIX
  • LCK_M_U
  • LCK_M_UIX
  • LCK_M_X
  • XACTLOCKINFO
