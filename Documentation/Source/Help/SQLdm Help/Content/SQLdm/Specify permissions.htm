Administration Permissions

Logins with Administration permissions can access all monitored SQL Server instances and modify all settings including grooming options, notification rules and providers, manage SQLdm licenses, manage application security including the ability to add custom counters, and has the ability to add and remove SQL Server instances in SQLdm.

    TIP

    You cannot edit or delete logins belonging to the sysadmin role on the SQLdm Repository as they have SQLdm Administrator privileges.
