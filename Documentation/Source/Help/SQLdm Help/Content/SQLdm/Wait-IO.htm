Wait type category - I/O Page Latch

The I/O Page Latch wait type category includes all the waits that are associated with non-I/O page latch bottlenecks, including the following:

  • ASYNC_IO_COMPLETION
  • IO_COMPLETION
  • PAGEIOLATCH_DT 
  • PAGEIOLATCH_EX
  • PAGEIOLATCH_KP
  • PAGEIOLATCH_NL
  • PAGEIOLATCH_SH
  • PAGEIOLATCH_UP
