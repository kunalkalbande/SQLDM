Disk Details

The Disk Details report lets you view key disk metrics of a monitored SQL Server instance and specific disk drive over a specified period of time. To successfully monitor OS metrics, make sure that WMI/OLE automation is enabled on the monitored instance. Check Show Tabular Data to display results in a single tabular or grid format.

When to run this report

You should run the Disk Details report to analyze the six key metrics for disk speed, such as disk time per read or disk reads per second.

How SQLdm calculates metric values on this report

Most values on the Disk Details report correlate with the related SQL Server metric.

However, some metric values represent averages over time. Likewise, a few metric values are based on performance algorithms that use multiple statistics to calculate a more accurate metric.

You can access our Customer Service Portal (www.idera.com/support) for more information about the algorithm used to calculate a specific metric.
