Select counter order

Update the counter order for this report to specify where you want SQLdm to display the counters included in your custom report and how to aggregate metrics.

SQLdm offers the following options to change the aggregation for each metric included in your report.

Maximum

Displays the maximum value for the metric since the last collection.

Average

Displays the average per second value since the last collection.

Per Minute

Displays the average per minute value since the last collection.

To rearrange or change the metric aggregation for your counters:

  1. If you want to rearrange a counter in your report, click the up or down arrow next to the counter you want to move.
  2. If you want to change the aggregation for a metric in your report, select the appropriate Aggregation method.
  3. Click Next.
  4. Complete the Custom Report wizard by clicking Finish.
