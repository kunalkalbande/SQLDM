PowerShell Cmdlets for SQLdm

The following cmdlets help you automate the administration of your SQLdm deployment.

Cmdlet Name

Description

Get-SQLServers

Retrieve a list of SQL Servers on the network

Escape-SQLdmName

Allows you to specify server names containing special characters, such as |\?/<>[]:*

Grant-SQLdmPermissions

Grant SQLdm permissions on a server

Revoke-SQLdmPermission

Revoke SQLdm permissions on a server

New-SQLdmMonitoredInstance

Add a SQL Server Instance that you want monitored by SQLdm

Set-SQLdmMonitoredInstance

Configure options on your SQL Server Instance

Remove-SQLdmMonitoredInstance

Remove a SQL Server Instance from SQLdm monitoring

Set-SQLdmAppSecurity

Enable or disable Application Security for SQLdm

New-SQLdmUser

Create a new SQLdm user

Set-SQLdmUser

Modify an existing SQLdm user

Remove-SQLdmUser

Remove an existing SQLdm user

New-SQLdmDrive

Create a drive for connecting to the SQLdm Repository
