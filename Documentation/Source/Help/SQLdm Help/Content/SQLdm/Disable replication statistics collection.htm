Disable replication statistics collection

The gathering of replication statistics can require a significant amount of system resources. The Replication tab of the Monitored SQL Server Properties window provides the option to disable the collection of replication statistics and conserve system resources.

Collecting replication statistics requires a large amount of system resources, especially when your publisher or distribution queues contain a large number of items. You can easily enable this feature when you need replication statistics, and then disable it once you've collected enough data.

Access the Replication tab

You can access the Replication tab of the Monitor SQL Server Properties window by right-clicking the appropriate monitored SQL Server instance, and then selecting Properties. Click Replication when SQLdm displays the Monitored SQL Server Properties window.
