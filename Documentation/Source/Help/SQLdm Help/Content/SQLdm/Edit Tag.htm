Edit a tag

The Tag Properties window allows you to view and change the properties of an existing Tag. To access the Tag Properties window, right-click a Tag and select Tag Properties.

The following information is available and editable on the Tag Properties window:

Servers

The Servers tab lists the SQL Server instances monitored by SQLdm and allows you to select or deselect the servers included in the tag.

Custom Counters

The Custom Counters tab allows you to choose the custom counters you want to include in your tag.

Application Security

The Application security tab allows you to select the application security permissions you want to apply to your tag.
