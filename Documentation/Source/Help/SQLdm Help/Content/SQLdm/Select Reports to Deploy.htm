Select reports to deploy

SQLdm includes built-in reports that are specially designed to generate commonly-requested metrics reports using the SQL Server data collected in the repository.

Select reports

In the Reports Deployment wizard check the box for each report you want to deploy to a Microsoft Reporting Services Server, then click Next.

Available Reports

You can generate reports from the following report categories:

Server Monitoring Reports

Report Name

Description

Enterprise Summary

View the health of your SQL Servers.

Server Summary

View the health details of a single SQL Server.

Active Alerts

View the active alerts for all monitored SQL Servers.

Mirroring Summary

View the health of your mirrored databases.

Metric Thresholds

View all metric thresholds for a server.

Availability Group Topology

View the current topology of your availability groups configuration.

Virtualization Monitoring Reports

Report Name

Description

View a summary of your virtualized environment.

Virtualization Statistics

Report Name

Description

Change Log Summary

Server Analysis Reports

Report Name

Description

Analyze and compare performance baselines across two SQL Server instances.
