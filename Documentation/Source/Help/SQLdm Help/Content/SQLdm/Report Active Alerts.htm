Active Alerts

The Active Alerts report lists all outstanding, active alerts that SQLdm has raised against your SQL Server instances. To see more detail about the health of a specific instance, click on that instance and the Server Summary report displays.

When to run this report

You should run the Active Alerts report to view all alerts currently active on your monitored instances. You can sort the alerts by severity level, so you can focus on the most critical alerts first.

How SQLdm calculates metric values on this report

The values on the Active Alerts report correlate with the related SQL Server metric.
