Mirroring Status alert

The Mirroring Status alert provides the status of the mirrored database based on the current databases status other than OK. By default, the values Synchronizing and Synchronized equal an OK state. A Suspended database triggers a Warning alert, and a Disconnected or Pending Failover database status triggers the Critical alert.

You can change the status for the mirrored databases in the Mirroring view of the Databases tab.

This alert is raised based on the diagnostic data collection interval specified in Server Properties. For more information, see Set general server options.
