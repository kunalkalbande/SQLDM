Database Read/Write Error Occurred alert

The Database Read/Write Error Occurred alert indicates that an I/O error occurred while attempting to read from or write to a database file. If this error persists consider the following solutions:

  • Replace the disks.
  • File may be corrupted, see if you can restore from a backup.

This alert is raised based on the diagnostic data collection interval specified in Server Properties. For more information, see Set general server options.
