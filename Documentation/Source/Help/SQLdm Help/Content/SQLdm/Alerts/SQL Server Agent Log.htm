SQL Server Agent Log alert

The SQL Server Agent Log alert indicates that specific text appeared in the SQL Server Agent error log. You can customize which text triggers this alert by using the Alert Filters tab on the Advanced Alert Configuration window. This tab also allows you to customize the severity of the alert based on the frequency of occurrence.

The default settings are:

  • 2 - Warning
  • 1 - Critical

Use a size limit on your SQL Server Agent log

SQLdm allows you to limit the size of your SQL Server Agent log to help improve response time and avoid using unnecessary space on your server. For additional information and steps on setting this limit, see Configure text and expression alerts.
