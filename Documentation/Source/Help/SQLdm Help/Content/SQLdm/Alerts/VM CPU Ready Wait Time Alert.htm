VM CPU Ready Wait Time (ms) alert

The VM CPU Ready Wait Time (MS) alert provides the time (in milliseconds) that a virtual CPU is waiting for a physical CPU to become available for processing for the virtual machine on which the monitored SQL Server resides. A spike in this value may indicate CPU pressure on the host server. This alert is enabled by default and is only relevant to instances hosted on VMWare vCenter virtual machines enabled for VM monitoring in the SQLdm VM Configuration window.
