Enterprise Summary

The Enterprise Summary report provides an overview of the health and performance of all SQL Server instances monitored by SQLdm. Click the appropriate instance and SQLdm displays the Server Summary report.

When to run this report

You can run the Enterprise Summary report to view a quick overview of the health and performance of all monitored instances. This report allows you to see the status of all your SQL Server instances at a glance.

How SQLdm calculates metric values on this report

Most values on the Enterprise Summary report correlate with the related SQL Server metric. However, some metric values represent averages over time. Likewise, a few metric values are based on performance algorithms that use multiple statistics to calculate a more accurate metric.
