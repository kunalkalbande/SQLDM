Wait type category - Memory

The Memory wait type category includes all the waits that are associated with memory bottlenecks, including the following:

  • CMEMTHREAD
  • RESOURCE_SEMAPHORE
  • RESOURCE_SEMAPHORE_MUTEX
  • RESOURCE_SEMAPHORE_QUERY_COMPILE
  • RESOURCE_SEMAPHORE_SMALL_QUERY
  • SOS_RESERVEDMEMBLOCKLIST
