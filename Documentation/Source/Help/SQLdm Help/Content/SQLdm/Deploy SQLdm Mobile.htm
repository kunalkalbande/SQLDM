Deploy SQLdm Mobile or Idera Newsfeed

SQLdm Mobile is a Web-based application that includes the Idera Newsfeed technology and several dashboard views that define your server health and expose critical alerts. You can access the SQLdm Mobile Web site from most smart phones and mobile devices. The Idera Newsfeed is available quickly as a tab on the SQLdm Today page.
