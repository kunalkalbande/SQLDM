Connect to the SQLdm Repository

The first time you open the SQLdm Management Console, SQLdm displays the Connect to SQLdm Repository window. Use this window to select the SQL Server instance, SQLdm Repository database name, and enter the authentication details the SQLdm Console uses to communicate with the SQLdm Repository.

The SQLdm Console requires Read Access on the SQLdm Repository database. You can use either Windows Authentication or SQL Server Authentication for this connection.

Access the Connect to SQLdm Repository window

You can open the Connect to SQLdm Repository window from the SQLdm Console menu bar by selecting File > Connect to SQLdm Repository. The ability to connect to another Repository is important in disaster recovery situations where you need to change the location of your SQLdm Repository.
